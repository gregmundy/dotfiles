# AeroSpace Keyboard Shortcuts

**Modifier:** `Ctrl+Opt` (⌃⌥) • **Move:** add `Shift` (⇧) • **Arrow keys:** direction

## Focus Windows

| Shortcut | Action |
|----------|--------|
| `Ctrl+Opt+Arrows` | Move focus (wraps around workspace) |

## Move Windows

| Shortcut | Action |
|----------|--------|
| `Ctrl+Opt+Shift+Arrows` | Move window in the layout |

## Workspaces

| Shortcut | Action |
|----------|--------|
| `Ctrl+Opt+1-5` | Switch workspace |
| `Ctrl+Opt+Shift+1-5` | Move window to workspace |
| `Ctrl+Opt+Tab` | Toggle to previous workspace |
| `Ctrl+Opt+Shift+Tab` | Move workspace to next monitor (wrap-around) |

## Layouts & Window State

| Shortcut | Action |
|----------|--------|
| `Ctrl+Opt+/` | Toggle tile split orientation |
| `Ctrl+Opt+,` | Toggle accordion layout |
| `Ctrl+Opt+F` | Toggle fullscreen |
| `Ctrl+Opt+Shift+Space` | Toggle floating / tiling |

## Resize Mode

| Shortcut | Action |
|----------|--------|
| `Ctrl+Opt+R` | Enter resize mode |
| `Left/Right` | Resize width |
| `Up/Down` | Resize height |
| `Enter/Esc` | Exit resize mode |

## Service Mode

| Shortcut | Action |
|----------|--------|
| `Ctrl+Opt+Shift+S` | Enter service mode |
| `R` | Reset / flatten layout |
| `F` | Toggle floating / tiling |
| `Backspace` | Close all but focused window |
| `Esc` | Reload config & exit |

---

**Note:** AeroSpace manages window structure. Raycast handles app launching.
